Windows XP、7、10通用的VBScript脚本

Posted on 2017-03-13 | In Windows

在需要给一批电脑打补丁时，虽然并不难，但比较耗时，使用脚本自动完成是最好不过的了。在Windowsw常用的是批处理脚本（.bat）,如果有高级的操作，可以用Powershell脚本，还可以用VBScript脚本。这些脚本在Windows XP、7、10这些主流版本上通用，这很重要，毕竟最好不要对别人的电脑作其他修改。只需在要打补丁的电脑上运行脚本即可完成操作，大大节省像修改Path和复制替换文件这些操作的时间。像C#这类可能更好用，但依赖于.NET framework，有些电脑不一定装得上，装的上亦不能保证不出问题，所以用一些通用的脚本还是很实用的。
